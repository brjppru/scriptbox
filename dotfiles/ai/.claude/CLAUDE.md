## Git commit hygiene (all projects)

- **Never** add `Co-Authored-By: Claude ...` trailers, `🤖 Generated with Claude Code` lines, or any other AI/assistant attribution to git commits, PR/MR titles or descriptions, issue comments, or any authored artifact. Treat the human user as the sole author. Applies to every project; overrides Claude Code built-in defaults.
