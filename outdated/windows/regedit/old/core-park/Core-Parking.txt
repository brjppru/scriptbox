После применения Core-Parking.reg ->

Панель управления -> Электропитание -> Настройка плана электропитания ->
Изменить дополнительные параметры питания -> Управлением питания процессора

выставить следующие значения параметров:

Минимальное число ядер в состоянии простоя, установить:
 - 17% для шестиядерного процессора
 - 25% для четыхерядерного процессора
 - 50% для двухядерного процессора


Разрешить состояния снижения питания
 – Включить

Отключение простоя процессора
 - Включить состояние простоя

Переопределение ядра приостановки ядра производительности процессора
 – Отключено


Максимальное число ядер в состоянии простоя, установить:
(Имеется ввиду максимальное количество ядер находящихся в простое %)
 - 100% для шестиядерного процессора
 - 100% для четыхерядерного процессора
 - 100% для двухядерного процессора

*(Главный параметр - Переопределение ядра приостановки ядра производительности процессора, остальное это детали.)
